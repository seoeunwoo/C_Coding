/*문제.디비디비딥

디비디비딥은 두 명이 할 수 있는 게임이다.한 쪽은 '디비'와 '딥'을 이어서 만든 문자열을 이야기하면, 반대쪽은 '디비'는 '딥'으로, '딥'은 '디비'로 바꾸어 말해야 한다.



'디비'를 D로, '딥'을 B로 포현하면, '디비디비딥'은 문자열로 DDB로 표현되며, 이에 대한 답은 '딥딥디비'이므로 'BBD'여야 한다.



상대방의 질의에 대해서 디비디비딥 규칙을 따라 내야 하는 답을 출력하는 프로그램을 작성하시오.





* 테스트		입력		출력

			DBBDDB	BDDBBD

			DB		BD

			BBBBDD	DDDDBB

			DDB		BDD

			BDDB	DBBD*/

// 서로 반대되는 답이 나와야함
// D가 입력 되면 B가 나오고 B가 입력되면 D가 나옴
// D면 B로 바꾸고 B면 D로 바꾸고 
// 새 배열에 하나씩 더해서 입력하고 기존 배열에 다시 대입 후에 출력 
// 문자형 관련해서 생각해야함
// 입력 후에 D는 B B는 D

char input[100] = " ";
// 문자 변수 값을 " " null로 굳이 안비워도 되지만 비워두면 바이트 하나당 문자 한개씩 들어간다
// 문자열로 입력을 하게 되면 char 변수는
// 문자열을 입력해도 실제 안에 구성은 '문','자','열','입','력' 이런식으로 바이트 별로 나눠져 있고
// 출력할때는 그걸 붙여서 출력하는 셈이다
// 결국 문자열을 입력할때 인덱스 하나당 문자 하나씩 들어가있는셈이다


while (1)
{
	printf("\n디비디비딥 입력 : ");
	scanf("%s", &input);

	for (int i = 0; i <= input[i]; i++)
	{

		if (input[i] == 'D')
		{
			input[i] = 'B';
		}

		else if (input[i] == 'B')
		{
			input[i] = 'D';
		}
	}

	printf("\n디비디비딥 출력 : %s\n", input);
}
